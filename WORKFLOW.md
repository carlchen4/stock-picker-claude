# 完整投资流程 (WORKFLOW)

把所有工具串成一个操作流程 + 已定原则。
（运行命令细节见 `README_run.md`；这份偏"决策+原则"。）

---

## 📅 每月（核心 · ~5 分钟）

```bash
./run.sh monthly        # 加拿大 + 美股一起跑，邮件到 carlchenn@hotmail.com
```
1. 收两封邮件（TSX + US Tech），每封 = 6–8 只等权持仓表（公司/行业/价格/股数）
2. **当天市价买入**（择时已证明负期望 → 不等、不挂限价抄底、不等"企稳确认"）
3. **加拿大若选中保险/资管（MFC/SLF/FFH/BAM/BN）→ 手动跳过**，换顺位银行或现金（"只投银行"偏好；模型仍跑全 12 只以保 alpha，执行时跳过）

---

## 📊 每季（银行财报季后 · 约 2 / 5 / 8 / 12 月底）

```bash
# 1. 更新银行手填数据（财报里的 CET1 / PCL / 分部），只改 CSV
#    编辑 bank_data_ca.csv
./run.sh banks                    # 7 银行基本面排名

# 2. 刷新基本面（自动）
./run.sh fund                     # 加拿大 30
./run.sh fund-us                  # 美股 21
python3 financials_score_ca.py    # 加拿大全 12 金融

# 3. 刷新风险+AI 报告（联网，照 runbook）
#    见 RISK_AI_REPORT_PROMPT.md（6 条搜索 query + 结构）
```

---

## 🧭 每月决策流程

```
模型出名单（动量为主）
   │
   ├─ 加拿大：选中保险/资管？→ 跳过，换银行/现金
   ├─ 交叉验证基本面：这只在 fundamentals 榜太差吗？
   │     （如 INTC/TSLA 基本面垫底 = 纯波动率押注，知道即可）
   ├─ 看风险报告：重大软肋？（财报日、信用恶化、估值雷）
   └─ 执行：当天市价、等权买入
```
**模型(动量)是主，基本面/风险是辅** —— 不推翻模型，只为知道"每只为什么被选、风险在哪"。

## ⚖️ 已定原则（别重复踩坑）

| 原则 | 原因 |
|------|------|
| **模型不再调** | 特征/调参/扩池/Triple ML(两版)/REIT 特征全试过，**都回归** → 局部最优 |
| **当天市价买，不择时** | 实测负期望：限价抄底贵 ~1%(CA)/~3%(US)；等确认=追高 |
| **不投保险/资管** | 个人偏好；移除后模型 IR 0.96→0.35，故保留全 12 只、执行时跳过 |
| **银行 AI 看 Evident，不看"嘴"** | 提及次数会骗人（Scotia 提 45 次但创新 #44 垫底） |
| **提升只靠攒 OOS** | 84 月小样本撑不起更复杂模型；攒 6+ 月实盘再验证回测 |
| **加拿大用 yfinance** | FMP/SimFin 免费层不支持 .TO |
